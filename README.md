# Jatin Dutt Portfolio

This is my personal portfolio website. I am Jatin Dutt, a B.Tech student and full stack developer with interest in DevOps, cloud, and building practical web applications.

The main purpose of this project is to show my skills, experience, projects, education, and contact details in one clean responsive website.

## About This Project

This portfolio is built using React and Vite. I created it as a simple professional website that can be used for job applications, internships, and sharing my work with recruiters or teams.

## Features

- Responsive design for mobile, tablet, and desktop
- About, experience, projects, skills, education, and contact sections
- Mobile-friendly menu
- Download CV option
- Contact links for email, GitHub, LinkedIn, WhatsApp, and phone
- Basic SEO files like `robots.txt` and `sitemap.xml`

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- HTML

## How To Run

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
